User story

As a user, I want to be able to navigate to notifications to filter new customers, Owners first purchase order, purchase order dispatched by system, RFQ sent to supplier, supplier accept RFQ, supplier reject RFQ, system generate a bill to client, client accept or reject bill, system generate invoice, client mark invoice complete or returned

Acceptance Criteria

The notifications icon in placed on the top bar Notification show once the user clicks “Notifications icon” The notifications icon will be changed to selected The notifications filtered by day, week and month The notification filtered active icon colorchanged to accent color The notifications will show the date & time of notifications The number of notifications will be showing on the total notifications and on each type on notifications The notification area color will be focused if user hover on it The notification can mark as ridden and change its color from active color to inactive gray color The notification redirect to action page The notifications be filter based on user selection “types” The notifications can be all marked as read The notifications load more notification by clicking see more buttonthe notifications shows scroll bar to view more notifications The notifications will show empty icon if there no any notifications
